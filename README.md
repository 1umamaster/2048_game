# 2048 Game

A classic 2048 puzzle game built with vanilla JavaScript, featuring a clean UI, real-time score tracking, and smooth game logic. The purpose of this project is to demonstrate advanced DOM manipulation, CSS grid-like layouts using tables, and complex state management in JS.

## Live Preview

🔗 [**Play 2048 Live**](https://1umamaster.github.io/2048_game/)

## Technologies Used

* **JavaScript (ES6+)** — Core game logic, state management, and event handling.
* **SCSS/CSS3** — Custom styling, Flexbox for centering, and dynamic tile coloring.
* **HTML5** — Semantic structure for the game board and controls.
* **CommonJS/ES Modules** — Modular code structure for game classes.

## Getting Started

Follow these instructions to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/1umamaster/2048_game.git
   cd 2048_game

2. **Install dependencies:**
   ```bash
   npm install

3. **Run the project locally:**
   ```bash
   npm start

## Features

* **Advanced Game Logic:** Full implementation of tile sliding and merging in four directions.
* **Dynamic UI Updates:** Real-time score calculation and tile color changes based on value.
* **Game Status Management:** Separate states for "Start", "Win" (reaching 2048), and "Game Over".
* **Responsive Design:** Perfectly centered game field with adaptive layouts.
* **Keyboard Support:** Full control using arrow keys with prevented page scrolling.
